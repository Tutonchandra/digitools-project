📌 Project Name

Digitools – AI Tools & Digital Services Platform

📝 Description

Digitools is a modern web application that showcases various AI tools and digital services in a clean and user-friendly interface.
Users can explore tools, view pricing, add tools to cart, and interact with a smooth and responsive UI.

The project is designed to demonstrate React component structure, state management, and modern UI design using Tailwind CSS and DaisyUI.

🚀 Live Website:

🔗 https://digitools-project.netlify.app/

🛠️ Technology Used:

⚛️ React JS
🎨 Tailwind CSS
🌼 DaisyUI
🧠 JavaScript (ES6)
🔔 React Toastify
🎯 Vite
🧩 Lucide React Icons
✨ Features

1️⃣ AI Tools Listing
Displays multiple AI tools in a responsive card layout
Shows price, status, and description
Clean and modern UI

2️⃣ Add to Cart System
Users can subscribe/add tools to cart
Duplicate tools are prevented
Toast notification shows success message

3️⃣ Payment & Remove System
Users can remove tools from cart
Total price calculation
Payment button clears cart
